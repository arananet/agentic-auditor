# GEO Agentic Auditor

An AI-powered website auditor designed to evaluate a domain's readiness for Generative Engines (LLMs, AI Agents, and modern search). 

Developed by **Eduardo Arana** and **Soda 🥤**.

## 🏗 Architecture

- **Frontend**: Modular Next.js components.
- **Services**: Business logic isolated in \`src/services\`.
- **Types**: Shared TypeScript interfaces.
